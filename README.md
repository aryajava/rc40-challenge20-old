# Rubicamp Batch 40

> Challenge 20

This BREAD project is Express JS application with EJS template engine.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

To install the dependencies, run the following command:

```bash
npm install
```

## Usage

To start the application, use the following command:

```bash
npm run dev
```
